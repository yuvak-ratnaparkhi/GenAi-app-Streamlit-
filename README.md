# My First Streamlit App

A Python-powered **Streamlit** app showcasing interactive widgets, charts, file uploads, and data visualization using **Pandas**, **NumPy**, **Matplotlib**, and **Plotly**.

---

## Features

- Interactive greeting with user input  
- Dynamic line and bar charts using randomly generated data  
- Sidebar navigation with image and video embedding  
- CSV file upload and display with Pandas  
- Text, Markdown, code, headers, and subheaders demonstration  
- Various input widgets: text input, text area, number input, sliders, selectbox, multiselect, radio buttons, checkboxes  
- Conditional display using checkboxes and radio buttons  
- Login form using Streamlit forms  
- Multi-column layout and expander widgets  
- Data visualization with **Matplotlib** and **Plotly**
